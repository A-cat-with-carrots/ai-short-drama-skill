## 故障排查

| 现象 | 应对 |
|------|------|
| 角色脸跳 | 在 prompt 加更多 ref 引用 + 强调「与参考图一致」3 次 |
| 服装跳 | 在 prompt 显式写当前服装颜色 + 款式 |
| 镜头不动 | 强调运镜词：`镜头慢推 / camera dolly in slowly` |
| 出现广告水印 | prompt 末尾再加一遍 `no watermarks, no logos, no ads` |
| 风格不一致 | 每段都重写完整视觉关键词 |
| 嘴型不对 | 加 `表情、嘴型、呼吸、台词严格同步` |
| 角色比例不对 | 加 `半身 / 全身 / 大特写` 明确画幅 |
| 角色站位错 | 在 prompt 里写 position（左/中/右）|
