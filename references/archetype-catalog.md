# 短剧角色 / 题材原型库

不是凭空设角色。短剧观众认的就是几个固定原型。每个角色都要明确属于一个原型，再加一点反差。

## 题材原型

### 女频核心

#### 1. 重生复仇流
- **公式**：女主含冤死 → 重生 → 提前布局 → 一个个干掉前世仇人
- **核心爽点**：上帝视角碾压 / 反向利用对方招数
- **对标**：《重生拒当冤大头》
- **变种**：重生 + 商战 / 重生 + 年代 / 重生 + 玄学

#### 2. 老年觉醒（2026 新热）
- **公式**：被忽视的奶奶/外婆 → 一个事件触发觉醒 → 反向控制全家
- **核心爽点**：长辈智慧碾压晚辈算计
- **对标**：《奶奶觉醒后全家破产了》《奶奶的绝地反击》
- **变种**：奶奶联合孙女 / 奶奶背后是大佬

#### 3. 神医 / 相师 / 命格
- **公式**：女主隐藏身份（神医/相师/灵异体质）→ 救人 / 算命 → 揭露阴谋
- **核心爽点**：一眼看穿别人不知道的真相
- **对标**：《女相师》
- **变种**：相师 + 重生 / 神医 + 战神

#### 4. 离婚追妻火葬场
- **公式**：男主嫌弃女主 → 离婚 → 女主开挂逆袭 → 男主追妻不得
- **核心爽点**：女主的"不要了"
- **对标**：《有本事别复婚》
- **变种**：和闺蜜联手 / 直接嫁给男主敌人

#### 5. 甜宠青梅
- **公式**：双向暗恋 → 误会 → 第三者插足 → 重新和好
- **核心爽点**：高浓度撒糖 + 醋意大爆发
- **对标**：《半糖青梅》
- **状态**：稳定但不再爆款

#### 6. 婆媳虐恋
- **公式**：婆婆刁难 → 女主隐忍 → 反击 → 婆婆服软
- **核心爽点**：当众打脸婆婆
- **对标**：《家里家外 2》

#### 7. 穿越当 XX（皇后 / 王妃 / 大小姐）
- **公式**：现代女主穿越 → 用现代知识降维打击 → 拿捏古代人
- **核心爽点**：信息差碾压
- **状态**：⚠️ 已饱和，必须做差异化（如：穿越后发现是反派 / 穿越要任务）

### 男频核心

#### 1. 都市脑洞 + 响指系统
- **公式**：男主获得系统（响一下指就 X）→ 用系统打脸 → 越用越强
- **核心爽点**：响指 = 爽
- **对标**：番茄都市脑洞天花板

#### 2. 都市修真种田
- **公式**：男主有修真背景 → 在都市低调种田 / 经营 → 偶尔露一手
- **核心爽点**：扮猪吃虎
- **对标**：番茄完读 Top3

#### 3. 战神归来 / 龙王
- **公式**：男主隐藏战神身份 → 被前妻/家族轻视 → 身份暴露 → 全场跪
- **核心爽点**："你说什么，我老公是？"
- **状态**：稳定经典

#### 4. 赘婿逆袭
- **公式**：男主入赘 → 受尽白眼 → 真实身份是 X → 翻盘
- **状态**：⚠️ 已严重同质化

#### 5. 悬疑民俗 / 风水 / 侦破
- **公式**：男主特殊职业 → 接案 → 民俗 / 风水反转 → 揭秘
- **状态**：上升期黑马

## 用法

阶段 1 选 1 个题材主原型 + 1 个副原型。
阶段 3 每个角色明确归类 1 个角色原型 + 1 个反差点。
不要超原型——观众是来看熟悉的爽，不是看你创新。

但**反差点必须有**——没反差的纯原型 = 套路 = 观众觉得"又是这个"。

例：
- 隐藏战神 + 反差：怕老婆怕到当场跪
- 重生女王 + 反差：唯独对一只猫展示真情
- 老年觉醒 + 反差：会刷短视频、追星
