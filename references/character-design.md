```markdown

- **原型**：<从 archetype-catalog.md 选>
- **反差点**：<一句话>
- **年龄 / 性别 / 身高 / 体型**：
- **职业 / 身份**：
- **背景故事**（≤ 100 字）：
- **角色弧光**（在 N 集中如何变化）：
  - 起点（第 1 集）：<状态>
  - 拐点（第 X 集）：<事件 → 状态变化>
  - 终点（最后 1 集）：<状态>
- **视觉母题**（一个反复出现的色彩 / 物件 / 动作 / 配饰）：
- **象征意义**（这个角色代表什么主题）：
- **标志性动作**（≥ 2 个）：
- **口头禅 / 经典台词**（≥ 3 句）：
- **服装表**：
  - 第 1-N 集：<日常装>
  - 第 N+1-M 集：<转折后装>
  - 关键场合：<礼服 / 战衣等>
- **即梦角色 ref 提示词**（中文）：
  > 详细描述供即梦生成 ref 图：年龄、五官、发型、服装、气质、参考明星脸（如有）
- **即梦角色 ref 提示词**（英文，可选 backup）：
  > English version for 备用
- **角色关系**：<和其他角色的关系>
```

### 角色弧光示例（女主：苏曼，重生复仇剧）

- 起点：被丈夫和闺蜜联手害死
- 拐点（第 3 集）：识破闺蜜假面 → 决定不告发，反向布局
- 拐点（第 12 集）：发现奶奶才是幕后 → 改变策略
- 终点：把所有人送上自己曾死的位置 → 自己抽身离开

### 视觉母题示例

- 苏曼的母题：**红色丝带**（每次决定下手前会摸一下颈上的红丝带）
- 男主的母题：**银色打火机**（焦虑时会反复打开关闭）
- 反派的母题：**鳄鱼皮包**（炫耀时会拍包）

母题作用：观众无意识中识别角色 + 镜头有重复元素剪辑

## 场景 bible

只收录出现 ≥ 3 集的场景。每个场景：

```markdown
### 场景 N：<场景名>

- **场景 ID**：scene_<编号>
- **场景描述**（≤ 100 字）：物理空间、装饰、氛围
- **出现集数**：EP01, EP05, EP10, ...
- **关键视觉元素**：标志性物件、色调、灯光
- **情感色彩**（这个场景反复出现时承载什么情绪）：
- **重复母题**（场景中反复出现的元素）：
- **即梦场景 ref 提示词**（中文）：
  > 详细描述供即梦生成场景四宫格 ref（4 个角度：全景 / 入口 / 关键陈设特写 / 窗外）
```

## 道具 bible

只收录有剧情推动 / 象征意义的。日常物件不收录。

```markdown
### 道具 N：<道具名>

- **道具 ID**：prop_<编号>
- **描述**（≤ 50 字）：外观、材质、尺寸
- **出现集数**：
- **剧情功能**：（如：解开身世之谜的钥匙）
- **象征意义**：（如：象征女主对母亲的执念）
- **即梦道具 ref 提示词**（中文）：
  > 详细描述供即梦生成道具三视图（正面 / 侧面 / 背面 / 细节特写）
```

## 角色 / 场景 / 道具数量上限

| 项 | 上限 | 理由 |
|----|------|------|
| 主要角色 | 4 | 短剧 30s 容不下更多脸 |
| 次要角色 | 2 | 用于反派 + 工具人 |
| 场景 | 5 | 每多一个就多一份成本 |
| 道具 | 5 | 多了观众记不住 |

超上限要砍。砍的原则：合并 → 删除 → 替换为文字交代。

## 角色关系图

在 `character_bible.md` 末尾画 ASCII 关系图：

```

       ╱         ╲

                   │
```

| 角色 A | 关系 | 角色 B | 起点 | 终点 |
|--------|------|--------|------|------|
| 苏曼 | 表面闺蜜 / 实际仇人 | 沈玉 | 信任 | 撕破 |
| 苏曼 | 假装恩爱 / 实际复仇 | 奶奶 | 孝顺 | 揭穿 |

## 即梦角色 ref 生成路径（顺手帮用户）

虽然本技能不调即梦 API，但要在 character_bible.md 末尾给出 **可复制粘贴到即梦的角色 ref prompt**：

```markdown
### 即梦角色参考图生成包

#### 苏曼_ref.png
**即梦输入**（使用「图像生成」模式 / 9:16 / Seedream 3.0）：

> 25 岁亚洲女性，鹅蛋脸，黑色长发披肩，眼神冷静含怒，穿米色高领针织衫 + 卡其色风衣，颈上系一条暗红色丝绸丝带（视觉母题），站立半身像，背景纯灰色，电影质感，浅景深，9:16 竖屏

#### 林策_ref.png
**即梦输入**：
> ...
```

这样用户拿到 character_bible.md 立刻可以批量生角色 ref，不用再问怎么写 prompt。
