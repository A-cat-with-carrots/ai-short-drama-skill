# 阶段 5：批判 refine 细则

## 为什么必做

一次出稿的短剧 = 烂剧。所有爆款短剧背后都是 3-10 轮打磨。

本阶段的目的不是赞美自己写的，而是**把它当作一个 0 分作品**重新审视。

## 触发时机

- 阶段 2（架构）做完后做一次小批判
- 阶段 3（设定圣经）做完后做一次小批判
- 阶段 4 做完前 3 集和最后 1 集后做一次大批判（写入 critique.md）
- 全部做完后做一次终批判

## 批判 6 维度（每维 0-10 分）

### 1. 黄金 3 秒钩子强度

测试方法：把第 1 集前 3 秒读给一个不知本剧的人听，看 ta 是否想看下去。

- 10：立刻问"然后呢"
- 8-9：感兴趣
- 6-7：将信将疑
- ≤ 5：会划走

打分标准：
- 有具体动作 / 冲突 / 反转 → +3
- 第 1 句话就是宣战 / 揭秘 / 极端动作 → +3
- 视觉冲击强（撞 / 砸 / 摔 / 血）→ +2
- 钩子是熟梗（套路化）→ -3
- 钩子需要 5s 以上才能 get → -5

### 2. 可预测度

测试方法：看完前 3 集，能猜中结局吗？

- 10：完全猜不到，但回头看伏笔都对
- 8-9：能猜大方向，细节意外
- 6-7：能猜大半
- ≤ 5：第 1 集就猜到了 → 致命问题

如果 ≤ 5：必须改。改的方向：
- 加一个反派的反派（让真正的幕后黑手不是观众第一直觉的那个）
- 男女主关系反转（从仇人变盟友 / 从盟友变仇人）
- 主角身份反转（最后揭穿其实是 X）

### 3. 同质化风险

测试方法：列出近 30 天同题材爆款的 3 部，逐项对比：

- 开场前 3 秒：是不是和爆款相似？
- 男女主关系：是不是同一种？
- 反转点：是不是同一种？

每和一部爆款相似一项 -1 分。

如果 ≤ 5：要么改差异化角度，要么挖一个独有的「视觉母题 + 角色反差」让用户记住。

### 4. 爽点密度

测试方法：数全集 satisfaction 数量。

| 集长 | 最低爽点数 | 满分爽点数 |
|------|-----------|------------|
| 30s | 1 | 2 |
| 60s | 2 | 4 |
| 90s | 3 | 5 |

打分：每少 1 个 -2 分，每超满分 1 个 -1 分（爽点过密会麻木）。

### 5. 角色辨识度

测试方法：把所有角色名换成 A/B/C，让人读对话，能猜出谁是谁吗？

每个角色必须有：
- 独特口头禅 → +1
- 独特动作 → +1
- 独特视觉母题 → +1
- 鲜明性格反差 → +2

满分 5。低于 3 → 角色没立住，回去重写人物 bible。

### 6. 付费点强度（仅付费短剧）

测试方法：第 10 集结尾的 paywall_punch 反转，强到能让人掏钱吗？

- 10：必须知道下一集发生什么，会立刻付费
- 8-9：很想知道，犹豫一下会付费
- 6-7：好奇但可能不付费
- ≤ 5：致命，付费率会扑街

打分标准：
- 第 10 集结尾是核心反转（不是过渡）→ +3
- 反转涉及主角身份 / 关系 / 命运 → +3
- 反转是前 9 集所有伏笔的兑现 → +2
- 反转留 cliffhanger 半句话 → +2

## 必改的判断

- 单维度 ≤ 5 → 必改
- 总分 < 40 → 大改（结构性返工）
- 平台审核任意一项不通过 → 必改
- 反同质化任意一项相似 → 必改

## 用户介入时机

写完 critique.md 后给用户看，让用户：

- 确认修订优先级（你给的优先级可能不对）
- 决定哪些建议采纳、哪些不采纳
- 提出额外修订点

然后**真的去改**——不是写完批判就结束，要在原文件上落地修订。

修订完后再做一次小批判，分数有没有上来。如果某项还是 ≤ 5，继续改。

## 批判态度

- 不要赞美自己写的（"这个钩子很好" 都不要写）
- 不要"中立"（"可以更好" 等于没说）
- 要尖锐：直接写"这个钩子是套路，会被划走"
- 要有具体改写方案：不要说"建议加强"，要说"把第 1 句改成 'XXX'"

如果你写的批判全是 8-10 分，说明你不敢真批判。回去用更狠的眼光再看一遍。
