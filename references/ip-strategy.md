# 阶段 1：IP 定位深度细则

## 7 个维度（写入 ip_brief.md）

### 1. 题材原型

从 `archetype-catalog.md` 选：

- **主原型**（决定核心走向）
- **副原型**（决定差异化）

例：
- 主：重生复仇 + 副：经商爽文 → 《重生拒当冤大头》风格
- 主：老年觉醒 + 副：家族阴谋 → 反套路差异化
- 主：神医相师 + 副：穿越 → 《女相师》风格

不要选 3 个原型——会失焦。

### 2. 目标受众

| 维度 | 选项 | 怎么选 |
|------|------|--------|
| 性别 | 女频 / 男频 / 全龄 | 红果 81.7% 女频，没特别理由就女频 |
| 年龄段 | 18-25 学生 / 25-40 都市女性 / 40+ 中老年 | 老年觉醒 → 40+；甜宠 → 18-30 |
| 平台 | 红果 / 抖音 / 快手 / 番茄 / TikTok 出海 | 影响时长和审核 |
| 付费意愿 | 免费（广告） / 付费（解锁） | 决定集数和付费点结构 |

**写法**：
```
受众：25-40 岁都市女性，红果 + 抖音双平台投放，付费短剧
```

### 3. 对标 IP（"X 的设定 + Y 的节奏"）

至少 2 部已爆短剧或小说。说明对标的具体维度（不要只说"对标 XXX"）。

例：
- "《女相师》的玄学设定 + 《重生拒当冤大头》的商战节奏 + 《奶奶觉醒》的反套路开场"
- "《赘婿》的爽点公式 + 《狂飙》的人物群像"

### 4. 核心钩子（一句话）

要求：

- ≤ 25 字
- 看到立刻想知道"然后呢"
- 自带矛盾或悬念
- 不能是设定，必须是行动 / 反转 / 冲突

❌ 反例：
- "讲一个穿越当皇后的故事"（这是设定，没钩子）
- "女主很厉害，男主追她"（没具体冲突）

✅ 正例：
- "女主穿越落地砸晕皇帝，发现自己是先皇宠妃"
- "奶奶在自己 80 大寿当天宣布全家断亲，理由 30 年没说出口"
- "重生后她做的第一件事，是给前世的'闺蜜'写讣告"

### 5. 重复爽点公式

定义"在每集都要复现的那个让观众嘶——爽的瞬间的公式"。

公式结构：`<触发条件> + <主角动作> + <对方反应>`

例（爽剧）：
- 触发：有人当众轻视女主 → 动作：女主用一句话点破对方致命弱点 → 反应：对方失态崩溃
- 触发：男主被旧情人挑衅 → 动作：男主当场宣布与女主婚讯 → 反应：旧情人崩溃

例（虐剧）：
- 触发：男主以为女主背叛 → 动作：男主放狠话羞辱 → 反应：女主默默承受不解释

每集都要至少出现 1 次这个公式。这就是观众"上瘾"的化学反应点。

### 6. 集数与定价

| 模式 | 集数 | 单集长度 | 付费点 | 适合题材 |
|------|------|----------|--------|----------|
| 免费短剧 | 24-30 | 30s-90s | 无（靠广告） | 沙雕、轻甜、入门测试 |
| 付费短剧 | 60-100 | 60s-120s | 第 10 集首付费，每 5 集一付费点 | 爽剧、虐剧、IP 改编 |
| 长剧条 | 16-24 | 5-8 分钟 | 平台分账 | 精品 IP |

**付费短剧的付费点结构**：

```
第  1- 9 集：免费（钩子拉满，建立沉没成本）
第 10 集：💰 首付费点 — 必须是该剧到此为止情绪最高的一集，结尾用强反转
第 11-15 集：付费 — 兑现第 10 集的反转，再埋新钩子
第 16 集：💰 第二付费点 — 反转 / 揭秘
... 每 5 集一个付费点
```

### 7. 基调与视觉

| 基调 | 表征 | 视觉 |
|------|------|------|
| 沙雕爽 | 段子密度高、表演夸张 | 高饱和、暖色、近景特写多 |
| 狗血爽 | 反转密度高、情绪极化 | 高对比、冷暖分裂、广角 |
| 治愈甜 | 节奏慢、对话多 | 柔光、暖色、浅景深 |
| 暗黑虐 | 情绪压抑、配乐沉重 | 低饱和、冷色、阴影重 |
| 玄学神秘 | 有玄学元素、命理符号 | 暗色调、烟雾、符号特写 |

**视觉风格关键词**（即梦 prompt 用）：

- 沙雕爽：`vibrant colors, sitcom lighting, exaggerated expressions, vertical 9:16, cinematic`
- 狗血爽：`high contrast, dramatic lighting, melodrama, vertical 9:16, telephoto compression`
- 治愈甜：`soft warm light, shallow depth of field, pastel palette, vertical 9:16`
- 暗黑虐：`low key lighting, desaturated, chiaroscuro, vertical 9:16`
- 玄学神秘：`misty, dim ambient, candlelight, mystical symbols, vertical 9:16`
