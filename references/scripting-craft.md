# 阶段 4A：逐集脚本写作细则

## 单集结构（30s / 60s / 90s 通用）

```
[0-3s]   冷开场（cold_open）
[3-...]  正文（按 beat_sheet 推进）
[最后5s] 卡点（cliffhanger）
```

## 对话原则

1. **每句 ≤ 15 字**（短剧节奏需要，长句子会拖）
2. **不要解释，靠动作**（"她生气了" → 她把茶杯摔了）
3. **每句话都要有目的**（推进剧情 / 揭示性格 / 制造冲突，三选一）
4. **潜台词 > 表面意思**（最好的台词表面平静，底下刀光剑影）
5. **不要内心独白**（短剧没空间，靠表情演）
6. **冲突双方对话不要超过 3 个回合**（再多就拖；要么有第三人插入，要么镜头切走）

## 节奏标记（在脚本中用方括号标）

| 标记 | 含义 |
|------|------|
| `[钩子]` | 这句话是钩子 |
| `[爽点]` | 这句话/动作触发爽点公式 |
| `[反转]` | 这里是 twist |
| `[虐点]` | 这里是 pain |
| `[卡点]` | 这里是 cliffhanger |
| `[伏笔]` | 这里埋 setup（注明 payoff_ep） |
| `[兑现]` | 这里兑现伏笔（注明 setup_ep） |

例：
> 苏曼摸了摸颈上的红丝带[母题]，看着林策：
> 「下个月 8 号，你妈过寿。」[伏笔→ EP12]
> 林策皱眉：「你怎么知道？」
> 苏曼笑了笑：「我什么都知道。」[钩子]

## BGM / 音效写法

不要写"激烈音乐""悲伤音乐"——给配乐师 / 即梦音效模型没用。

写**具体音效配方**：

| 情绪 | 写法 |
|------|------|
| 爽点 | 鼓点猛击 + 反向音效 + 高频 hi-hat 渐入 |
| 虐点 | 钢琴单音 + 低频弦乐持续 + 80bpm |
| 钩子 | 心跳声 + 突然静音 + 脚步声 |
| 反转 | 唱片刮擦音效 + 0.5s 静音 + 鼓点 |
| 揭秘 | 弦乐渐强 + 风铃 + 镜头长焦 |

## 冷开场 5 种写法（再列一次，因为太重要）

1. **悬念问句**："你知道吗，我老公昨天死了，今天回来了"
2. **极端动作**：砸杯子 / 扇耳光 / 撞车
3. **反差画面**：豪车 + 猪圈
4. **意外反转**：3 秒内一个翻转
5. **直白宣战**：第一句就是宣战

## 卡点 3 种写法

1. **真相揭露半句**："其实你不是我——" 黑屏
2. **危机即将发生**：刺客举刀 → 黑屏
3. **新人物登场**：陌生人推门 → 给脸 → 黑屏

## 单集自查

- [ ] 0-3s 是否能让陌生观众想看下去？
- [ ] 全集对话总字数 ≤ 集长（30s ≤ 90 字、60s ≤ 180 字、90s ≤ 270 字）
- [ ] 每句台词 ≤ 15 字
- [ ] 没有内心独白
- [ ] 节奏标记齐全
- [ ] BGM 写到具体音效配方
- [ ] 卡点是 3 种万能模板之一
