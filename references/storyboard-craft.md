# 阶段 4B：分镜 + 镜头清单细则

## 9 宫格分镜（默认 30s 集长）

| 集长 | 结构 |
|------|------|
| 30s | 上 9 格（15s）+ 下 9 格（15s）= 18 格 |
| 60s | 上 18 格 + 下 18 格 = 36 格（或 4×9 格） |
| 90s | 6×9 格 = 54 格 |

每格 ≈ 1.67s，刚好对应即梦/Seedance 一段视频生成单元。

## storyboard.json 结构

每集一份 `storyboard.json`，包含 `part_a` + `part_b`，每 part 包含 9 格 grid。

详见 `references/schemas.md`。

每格 grid 必含 14 个字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `grid_number` | int | 1-9 |
| `time_start` | float | 秒 |
| `time_end` | float | 秒 |
| `scene_description` | string | 50 字画面描述 |
| `camera.type` | enum | 远景/中景/近景/特写/大特写 |
| `camera.movement` | enum | 固定/推/拉/摇/移/跟/升/降 |
| `camera.angle` | enum | 平视/俯视/仰视/正面/侧面/背面 |
| `characters` | array | 出场角色 + 动作 + 表情 + 位置 |
| `dialogue` | object | speaker + text + emotion，无对话则 null |
| `atmosphere` | string | 本格氛围 |
| `sfx` | string | 音效配方 |
| `bgm_change` | bool | 这格是否有 BGM 切换 |
| `continuity_flags` | array | 连戏标记（服装/道具/位置） |
| `jimeng_prompt` | string | 已拼好的即梦中文 prompt（核心交付物）|

## 镜头语法（每格选）

### 景别

| 景别 | 用途 | 短剧典型用法 |
|------|------|-------------|
| 大特写 | 五官 / 物件 | 眼神戏、关键道具特写 |
| 特写 | 头肩 | 对白、表情 |
| 近景 | 半身 | 对白、肢体动作 |
| 中景 | 膝以上 | 走动、互动 |
| 全景 | 全身 + 环境 | 场景建立、群戏 |
| 远景 | 大场景 | 环境氛围 |

**短剧默认偏向**：特写 + 近景为主（手机竖屏看小，远景看不清），关键时刻穿插全景。

### 运镜

| 运镜 | 表达 |
|------|------|
| 固定 | 稳定、客观 |
| 推 | 强调 / 进入情绪 |
| 拉 | 释放 / 揭示更大场面 |
| 摇 | 跟随注视 |
| 移 | 平行跟随 |
| 跟 | 跟随人物 |
| 升 | 拔高、揭示 |
| 降 | 压抑、降临 |

### 角度

| 角度 | 表达 |
|------|------|
| 平视 | 平等 |
| 俯视 | 弱势 / 渺小 |
| 仰视 | 强势 / 崇高 |

## 节奏镜头建议（按 beat 类型）

| beat 类型 | 推荐景别 | 运镜 |
|-----------|----------|------|
| cold_open | 特写 + 推 | 强冲击 |
| satisfaction | 中景 + 慢推 | 让爽点停留 |
| twist | 近景 + 突然推近 | 抓表情变化 |
| pain | 大特写 + 固定 | 让眼泪/血迹定格 |
| cliffhanger | 全景 + 拉 + 黑屏 | 留余韵 |

## 连戏标记（continuity_flags）

每格记录这格的"易出错连戏点"，方便剪辑回查：

```json
"continuity_flags": [
  "苏曼: 米色针织衫 + 红丝带（左肩）",
  "林策: 银色打火机（右口袋）",
  "桌上: 一杯咖啡（半满）"
]
```

## 即梦 jimeng_prompt 拼接规则（核心交付！）

每个 grid 的 `jimeng_prompt` 必须按这个模板拼：

```
[视觉风格关键词]，[景别 + 运镜 + 角度]，[场景描述]，[角色描述含 (@角色名_ref.png) 引用 + 动作 + 表情]，[氛围]，[同一角色，服装/发型/外貌不变]，9:16 竖屏，cinematic，电影感
```

实例（苏曼第 1 集第 1 格）：

```
高对比戏剧光影，米色暖调 + 暗红点缀，特写慢推，俯视，
古色古香的中式书房，案几上一支毛笔倒下，
(@苏曼_ref.png) 苏曼坐在案几后，左手摸颈上红丝带（视觉母题），右手握毛笔悬在半空，眼神冷静而锋利，嘴角微微上扬，
氛围：暴风雨前的平静，烛光摇曳，
同一角色，苏曼服装/发型/外貌不变，
9:16 竖屏，cinematic，电影感，浅景深
```

### 字幕

如有字幕（即梦支持），用中文括号包：

```
（"我不嫁。"）
```

不支持字幕的镜头不写。

### 角色 ref 引用

每提到一次角色，前面必加 `(@角色名_ref.png)`，无论该角色这格是不是主角。这样即梦的「全能参考」模式才能精确锚定。

### 视觉风格后缀必带

每段都要重写视觉风格关键词，因为即梦每段独立生成，不重写就跳风格。

## shot_list.md（人类可读表格）

每集一份，让导演/制片一眼扫完：

```markdown
# 第 1 集 镜头清单

## Part A（0-15s）

| 镜号 | 时间 | 景别 | 运镜 | 角度 | 角色 | 动作 | 对话 | 氛围 | 即梦 prompt 简版 |
|------|------|------|------|------|------|------|------|------|-----------------|
| 1 | 0.0-1.67 | 特写 | 慢推 | 俯视 | 苏曼 | 摸红丝带，握笔 | — | 暴风雨前 | 苏曼 + 红丝带 + 烛光 |
| 2 | 1.67-3.33 | 近景 | 固定 | 平视 | 苏曼 | 抬眼看镜头 | 「下个月 8 号...」[伏笔] | 冰冷 | 苏曼直视 + 烛光摇 |
| 3 | 3.33-5.0 | 大特写 | 推 | 仰视 | 林策 | 皱眉 | 「你怎么知道」 | 紧张 | 林策皱眉 + 阴影 |
| ... | | | | | | | | | |

## Part B（15-30s）

[同上]
```

## jimeng_batch.md（一键批量生成包）

每集一份，把 18 格的 `jimeng_prompt` 顺序排出来，每格之间用 `---` 分隔，方便用户在即梦逐段贴：

```markdown
# 第 1 集 即梦批量生成包

## 上传 ref 图清单
- 苏曼_ref.png
- 林策_ref.png
- scene_01_ref.png
- prop_01_ref.png

## Part A（共 9 段，每段约 1.67s）

### 段 1
> [完整 jimeng_prompt]

参数：9:16 / Seedance 2.0 / 全能参考 / 时长 2s

---

### 段 2
> [完整 jimeng_prompt]

参数：...

---

[... 9 段]

## Part B
[... 同结构 9 段]

## 拼接说明

1. 在剪映新建 9:16 项目
2. 按段 1 → 段 18 顺序导入
3. 段间用「叠化 0.3s」过渡
4. 配音用 [TTS / 真人]，配乐 [关键词]
5. 添加字幕：[关键台词列表]
```
